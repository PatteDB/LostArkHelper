# LostArkHelper

Small tool to help with Lost Ark stuff like honing, materials and raid planning.

Nothing fancy, just something I built to make things easier.


## What it does

- Fusion calculator (materials → fusion)
- Auction helper (best bid values)
- Material profit optimizer
- Character tracker (gear + GS + raids)
- Raid cheatsheets (zoomable)


## How to use

Just download the `.exe` from the Releases section and start it.

No install needed.


## First start

Windows might show a warning like:

"Windows protected your PC"

That’s normal because the app is not signed.

Just click:
More info → Run anyway


## Notes

- Everything is saved automatically
- Gearscore updates on its own
- Cheatsheets can be zoomed with mouse wheel and moved with drag
